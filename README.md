## О проекте

### Концепция разработки

Создание личного веб-ресурса требует баланса между визуальной эстетикой и технической сложностью. Проект FullStackForge разработан как масштабируемая платформа, объединяющая возможности серверного рендеринга и облачных баз данных в реальном времени.

- **Производительность:** Оптимизация Core Web Vitals с помощью Next.js App Router.
- **Безопасность:** Защищенная аутентификация администратора для управления контентом.
- **Масштабируемость:** Гибкая структура базы данных Firestore для динамического добавления кейсов.
- **UX/UI:** Плавные интерфейсные переходы и адаптивность под любые устройства.

### Решение: FullStackForge

Проект представляет собой не просто статичную страницу, а полноценное веб-приложение (SPA/SSR) с интегрированной панелью управления. Это позволяет обновлять портфолио без необходимости вносить правки в программный код и перезапускать сборку.

- **Динамическое управление данными:** Полная синхронизация с Firestore.
- **Безопасная авторизация:** Реализация NextAuth.js v5 с использованием Bcrypt.
- **Интерактивный интерфейс:** Использование Framer Motion для глубокого погружения пользователя.
- **Доступность:** Соответствие стандартам WAI-ARIA благодаря Radix UI.

## Технологии

### Frontend

- **Next.js 16.1.6 (App Router):** Основной фреймворк для гибридного рендеринга.
- **TypeScript:** Строгая типизация для минимизации runtime-ошибок.
- **Tailwind CSS:** Проектирование интерфейса на основе утилит.
- **Framer Motion:** Реализация декларативных анимаций.

### Backend & Infrastructure

- **Firebase Firestore:** NoSQL решение для хранения данных в реальном времени.
- **NextAuth.js v5:** Система управления сессиями и правами доступа.
- **Vercel:** Хостинг с поддержкой автоматизированного CI/CD.
- **Pnpm:** Эффективный менеджер пакетов для сокращения времени сборки.

## Возможности

- **Admin Dashboard:** Закрытый интерфейс для редактирования проектов и личной информации.

## Установка и запуск

Для работы проекта необходимо наличие установленного Node.js и менеджера пакетов pnpm.

1.  **Клонирование репозитория:**

    ```bash
    git clone https://github.com/S-NOWNUM-B/-FullStackForge-.git
    cd fullStackForge
    ```

2.  **Установка зависимостей:**

    ```bash
    pnpm install
    ```

3.  **Локальный запуск:**
    ```bash
    pnpm dev
    ```
    Приложение будет доступно по адресу: `http://localhost:3000`

## Деплой на Vercel

Проект оптимизирован для развертывания на Vercel. При пуше в ветку `main` деплой происходит автоматически.

Для ручного деплоя используйте Vercel CLI:

```bash
vercel deploy --prod
```
