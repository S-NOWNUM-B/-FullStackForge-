## Коротко обо мне

Я - Fullstack Developer и студент Software Engineering.
Разрабатываю масштабируемые приложения полного цикла: от архитектуры и работы с данными до продуманных интерфейсов.
Фокусируюсь на практичных решениях, чистой структуре проекта и удобстве для пользователя.

## Мой подход

Не просто написать код, а собрать цельный продукт: понятный, аккуратный и полезный людям.

- Думаю архитектурно, чтобы проект было легко развивать.
- Держу фокус на UX, чтобы интерфейс был приятным с первого экрана.
- Уделяю внимание деталям, которые формируют качество.

## Сейчас в фокусе

- развитие продуктового мышления в разработке;
- качество интерфейсов и визуальная чистота;
- стабильная и удобная админка для управления контентом.
